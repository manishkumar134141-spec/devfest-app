# ReviewShield 🚀

A brief description of what this project does. (e.g., *A web application built for DevFest that analyzes e-commerce product URLs to extract review/rating .*)

## ✨ Features
* Analyzes product URLs from platforms like Flipkart and Ajio.
* Analyzes product URLs from platforms like Flipkart and Amazon.
* Detects fake, suspicious, and genuine reviews
* Analyzes generic and repetitive text
* Provides a fake review percentage score
* Shows basic statistics and review breakdown
* Easy-to-use and simple interface

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS (Sora & Space Mono fonts)
* **Backend:** Node.js , Express.js

## 💻 Getting Started

Follow these instructions to get a copy of the project running on your local machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/manishkumar134141-spec/ReviewShield](https://github.com/manishkumar134141-spec/ReviewShield)

2. **Navigate into the directory:**
   ```bash
   cd reviewshield

3. **Install the dependencies:**
   ```bash
   npm install

4. **Run the server:**
   ```bash
   node server.js

5. **Open the app:** Open your browser and visit `http://localhost:5000` (or whatever port your server is running on).
